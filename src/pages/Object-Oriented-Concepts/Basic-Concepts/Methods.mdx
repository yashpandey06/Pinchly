## <strong style={{ color: "#ec407a" }}> What are methods ?</strong>



In Object-Oriented Programming (OOP), methods define the actions that objects can perform. Similar to functions, methods can take arguments and return values, enabling objects to interact with each other and perform complex tasks within a program.

Methods improve code readability and reusability i.e we write a method once and use it many times. We do not require to write code again and again. They can be classified as private, public, default, and protected, each providing different levels of access control.
We will cover these concepts in the upcoming lessons.
