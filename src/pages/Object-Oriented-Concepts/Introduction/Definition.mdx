## <strong style={{ color: "#ec407a" }}>Introduction</strong>

On this page, we'll learn the basics of Object-Oriented Programming (OOP). OOP includes ideas like inheritance, data binding, and polymorphism.

Simula was the first object-oriented programming language. In a truly object-oriented programming language, everything is an object.

## <strong style={{ color: "#ec407a" }}> What is OOPs ?</strong>

An object is a real-world thing like a pen, chair, table, computer, or watch. Object-Oriented Programming (OOP) is a way to design a program using classes and objects. It makes software development and maintenance easier by offering certain concepts:
