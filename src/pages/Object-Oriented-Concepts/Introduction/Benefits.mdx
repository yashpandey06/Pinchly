

## <strong style={{ color: "#ec407a" }}>Benefits of Object Oriented Programming</strong>
<br/>
<ul style={{ listStyleType: "disc", paddingLeft: "20px" }}>
  <li> OOP makes development and maintenance easier. In contrast, managing code in a procedure-oriented programming language becomes challenging as the project size increases. </li>
  <br/>
  <li> OOP provides data hiding, whereas, in a procedure-oriented programming language, global data can be accessed from anywhere.</li>
  
</ul>
